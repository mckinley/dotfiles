# dotfiles

## Install dotfiles
- `cd` into the dotfiles project root.
- run `./.dotfile-resources/bootstrap.sh`.

## Revert dotfiles
- `cd` into the dotfiles project root.
- run `./.dotfile-resources/bootstrap.sh --revert`.
